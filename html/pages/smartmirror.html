Smart Mirror

it’s pretty cool to see daily tasks, weather reports, latest news headlines, incoming messages and more while getting ready in front of the mirror — especially in our daily life with busy schedules. That’s how the Magic Mirror was born.

Learned how does small embeded systems work,gained knowledge on different kinds of Sensors,Understood about the API functionality and lot more take awaysa from this project

My prototype design before implementing the project

This Project has so many sub projects which to a whole made the Smart Mirror

Spotify Module

After installing the module and configuring and setting up the Spotify app the module displays the song you are currently listening to on Spotify. It shows on which of your devices you’re playing the song. If you like you can also display the album cover. To be able to display your currently playing song the module must connect to the Spotify service,achieved using Spotify API

Quote Module

Show a random quote from an online collection of quotations by authors, celebrities, and newsmakers.

News Headlines Module

Displays the Headlines on that day with the help of newapi.og API

Google Tasks and calender Remainder Module

Displays tasks from Google Tasks App and Calender Events which had been scheduled in our Mobile or other Devices with Google Account
